# 弹窗为该章节中所有的题目，与编辑模式的页面区别在于，若有阅读题，得把阅读题的小题也展示出来，且得有层级展示区分；

# 总分：下面每一级题的总分之和，根据设分，实时变化；

# 设分模式：

    · 每空平均分：根据题中空的个数平均给分；

    · 每空自定义：选择此项后，该题会出现总分，可点击，出现弹窗设置该题所有空的分数，未设置分数时，显示“未设分”，设置分数以后显示“XX分”，设置弹窗如下

# 该题总分：所有空分数之和；

# 空列表：根据空个数进行排布，过多支持弹窗内滚动；

# 交互：

    · 默认分值为空（不为0）；

    · 若有空未设分，点击确定，在弹窗内提示“还有未设分的空”；

    · 正常点击确定提交以后，主干题即显示该题总分；
